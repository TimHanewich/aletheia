API Service Pricing

API service is available at three service levels: Tier One, Tier Two, and Tier Three. We try our best to provide as much value as possible with generous allotments and endpoint availability for all tiers. As a result, Tier One, Two, and Three all offer identical endpoint access*; nothing is "off limits" for any tier.

Please note that the Aletheia API service is under continuous development. New services and endpoints will be available in the future that may be accessible only to the non-free Tiers due to demanding computation requirements.
